Which row breaks first?

Second row breaks first. Bootstrap supports upto 12 Columns for sm,md,lg and xl devices. Columns will rearrange automatically when we change the size of the screen. It is because md -medium device-width supports 768px or more, whereas sm-small device supports 576px or more. So, when the screen size is reduced, if the column px-size size reduced it goes to the next line.
